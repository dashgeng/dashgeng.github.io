### 基础知识

#### 常量和变量
* 定义：常量和变量把一个名字和一个指定类型的值关联起来。
* 常量的值一旦设定就不能改变，而变量的值可以随意更改。
* 常量和变量的使用原则是：先声明，后使用。
* 用 let 声明常量。

``` Swift
let pi = 3.141592654
```

* 用 ver 声明变量。
``` Swift
var currentTime = "12:06"
```

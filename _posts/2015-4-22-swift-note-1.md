### 基础知识

#### 常量和变量
* 常量和变量把一个名字和一个指定类型的值关联起来。

``` Swift
let Name : String
var Height : Int
```
* 常量的值一旦设定就不能改变，而变量的值可以随意更改。
* 选择常量还是变量：将不需要改变的值设置为常量，将需要改变的值设置为变量。
* 常量和变量的使用原则是：先声明，后使用。
* 用 `let` 声明常量。

``` Swift
let pi = 3.141592654
```

* 用 `ver` 声明变量。

``` Swift
var currentTime = "12:06"
```
* 在一行中可以同时声明多个常量或变量，之间用逗号`，`隔开。

``` Swift
let a = 3, b = 4, c = 5
var x = 10, y = 20, z = 15
```
